📚 Guia de Referência — Git & GitHub

Arquivo de exemplo para o curso técnico de Desenvolvimento de Sistemas

1. Configuração Inicial

ComandoDescrição
git config --global user.name "Seu Nome"Define o nome do usuário no Git
git config --global user.email "email@exemplo.com"Define o e-mail do usuário no Git
git config --listExibe as configurações atuais

2. Criação e Clonagem de Repositório

ComandoDescrição
git initInicializa um repositório Git na pasta atual
git clone <URL>Clona um repositório remoto para a máquina local

3. Controle de Versão — Ciclo Básico

ComandoDescrição
git statusMostra o estado atual dos arquivos
git add <arquivo>Adiciona um arquivo específico para o próximo commit
git add .Adiciona todos os arquivos modificados
git commit -m "mensagem"Salva um ponto de versão com uma mensagem descritiva
git logExibe o histórico de commits
git log --onelineExibe o histórico resumido em uma linha por commit

4. Repositório Remoto (GitHub)

ComandoDescrição
git remote add origin <URL>Conecta o repositório local ao repositório remoto
git remote -vExibe os repositórios remotos configurados
git push -u origin mainEnvia os commits para o GitHub (primeiro envio)
git pushEnvia novos commits para o GitHub
git pullBaixa e integra as alterações do repositório remoto

5. Branch e Merge

ComandoDescrição
git branchLista as branches existentes
git branch <nome>Cria uma nova branch
git checkout <nome>Muda para a branch especificada
git checkout -b <nome>Cria e já muda para a nova branch
git merge <nome>Une as alterações da branch especificada à branch atual
git branch -d <nome>Remove a branch após o merge
